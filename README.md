# 🎭 Darshita’s Modes — Interactive Portfolio

An immersive, multi-experience portfolio that transforms a traditional resume into an interactive product.

Instead of a static page, this portfolio allows users to explore my work through storytelling, gamification, and AI-driven interaction — demonstrating both technical depth and creative thinking.

## 💡 Concept

Most portfolios are passive.

This one is **interactive, experiential, and user-driven**.

It answers a simple question:
> What if a recruiter could *experience* a candidate instead of just reading about them?

---

## 🎯 Key Features

### 🎓 Classic Mode
- Structured professional portfolio
- Experience, skills, and project highlights
- Clean and recruiter-friendly layout

### 📖 Story Mode
- Cinematic vertical storytelling experience
- Presents journey, growth, and personality
- Focus on narrative-driven design

### 🎮 Play Mode
- Gamified interface with XP, levels, and exploration
- Makes portfolio engaging and memorable
- Demonstrates creative UI/UX thinking

### 🤖 AI Interview Mode
- AI-powered chat simulating a real interview
- Users can ask questions about my experience
- Built using LLM integration

---

## 🛠 Tech Stack

- **Frontend:** React 19, Vite, React Router v7  
- **Language:** TypeScript  
- **Styling:** Tailwind CSS v4, Shadcn UI  
- **Animations:** Framer Motion  
- **3D:** Three.js  
- **Backend & DB:** Convex  
- **Authentication:** Email OTP (Convex Auth)  
- **AI Integration:** OpenRouter (OpenAI SDK)  
- **Email Services:** Resend
